# 🎯 AI Resume Matcher

An AI-powered web app that analyses your resume against a job description and gives you an instant match score, highlights your strengths, identifies skill gaps, and suggests improvements.

Built with Python, Django, and Groq AI.

🌐 **Live Demo:** https://resume-matcher-u82y.onrender.com

---

## Features

- Paste your resume and any job description
- Get an instant match score out of 100
- See your top 3 strengths that match the role
- Identify missing skills and gaps
- Get specific suggestions to improve your resume

---

## Tech Stack

- **Backend:** Python, Django
- **AI:** Groq API (Llama 3.3 70B)
- **Frontend:** HTML, CSS, JavaScript
- **Version Control:** Git, GitHub

---

## Setup Instructions

1. Clone the repository
   git clone
   https://github.com/hidamusthafa/Resume-Matcher.git
   cd Resume-Matcher
2. Create a virtual environment
  python -m venv venv
  venv\Scripts\activate
3. Install dependencies
   pip install -r requirements.txt
4. Add your Groq API key in `matcher/views.py`
5. Run the server
   python manage.py runserver
6. Open `http://127.0.0.1:8000` in your browser
